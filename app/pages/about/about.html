About ionic-recorder

Version: {{version}}

ionic-recorder is an open source app, built with the Ionic2 / Angular2 frameworks, that allows you to record audio and organize your recordings. It works online or offline, on any smartphone with a recent browser.

For developers: this app demonstrates the browser's WebAudio API and IndexedDB built-in libraries.

This is a web-app, meaning it runs in a browser by visiting a web page, it does not require installation.
